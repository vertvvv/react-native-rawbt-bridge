# react-native-rawbt-bridge

RawBT bridge for react native

## Installation

```sh
npm install react-native-rawbt-bridge
```

## Usage

```js
import RawbtBridge from "react-native-rawbt-bridge";

// ...

const result = await RawbtBridge.multiply(3, 7);
```
